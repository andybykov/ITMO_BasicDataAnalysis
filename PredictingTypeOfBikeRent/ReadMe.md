### Описание

Учебный проект по машинному обучению, разработанный на платформе [ML.NET](https://ml.net/).

 Проект демонстрирует полный цикл создания модели машинного обучения для предсказания типа аренды велосипеда (краткосрочная или долгосрочная) на основе погодных и временных условий.



### Основные признаки

| Признак              | Описание                                   |
| -------------------- | ------------------------------------------ |
| **Season**           | Сезон года (зима, весна, лето, осень)      |
| **Month**            | Месяц (1-12)                               |
| **Hour**             | Час дня (0-23)                             |
| **Holiday**          | Праздничный день (да/нет)                  |
| **WeekDay**          | День недели (0-6)                          |
| **WorkingDay**       | Рабочий день (да/нет)                      |
| **WeatherCondition** | Погодные условия (ясно, туман, дождь/снег) |
| **Temperature**      | Температура воздуха (°C)                   |
| **Humidity**         | Влажность воздуха (%)                      |
| **WindSpeed**        | Скорость ветра (км/ч)                      |

**Целевая переменная:** `RentalTerm` — тип аренды (0 = краткосрочная, 1 = долгосрочная)



### Функциональные возможности

#### 1. Загрузка и анализ данных

- Загрузка данных из CSV-файла

- Исследовательский анализ данных (EDA)

- Визуализация распределения признаков

- Статистический анализ числовых данных

#### 2. Предобработка данных

- One-Hot Encoding категориальных признаков

- Нормализация числовых признаков (MinMax)

- Создание вектора признаков `Features`

- Разделение на обучающую (80%) и тестовую (20%) выборки

#### 3. Обучение и сравнение моделей

- **Logistic Regression (SDCA)** — быстрая линейная модель

- **FastTree** — градиентный бустинг на деревьях

- **LightGBM** — высокопроизводительный градиентный бустинг

#### 4. Оценка качества моделей

- Accuracy (точность)

- AUC (Area Under ROC Curve)

- F1 Score (гармоническое среднее)

- Precision и Recall для каждого класса

- Матрица ошибок (Confusion Matrix)

#### 5. Сохранение и загрузка модели

- Сохранение обученной модели в ZIP-файл

- Загрузка модели для повторного использования

- Предсказание на новых данных

#### 6. Демонстрация предсказаний

- Набор предопределённых тестовых сценариев

- Вывод вероятности предсказания

- Интерпретация результатов
