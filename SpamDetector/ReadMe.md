### Описание

Учебный проект по машинному обучению, разработанный на платформе [ML.NET](https://ml.net/).

Проект демонстрирует полный цикл создания модели классификации текста для определения спам-сообщений электронной почты на основе содержания письма.

### Основные признаки

| Признак     | Описание                |
| ----------- | ----------------------- |
| **Subject** | Тема письма             |
| **Content** | Содержание/текст письма |

**Целевая переменная:** `IsSpam` — тип сообщения (0/false = не спам, 1/true = спам)

### Функциональные возможности

#### 1. Загрузка и анализ данных

- Загрузка данных из CSV-файла с размеченными сообщениями
- Просмотр структуры набора данных
- Статистический анализ распределения классов

#### 2. Предобработка данных

- Текстовая нормализация и очистка
- Извлечение признаков из текста (Featurization)
- Кросс-валидация (перекрестная проверка) для оценки стабильности модели

#### 3. Обучение модели

- **TextClassificationMulti** — тренер многоклассовой классификации текста на основе предобученной модели BERT (Microsoft Research). 
  
  Для бинарной классификации (спам/не спам) используется как частный случай многоклассовой.

#### 4. Оценка качества модели

- MacroAccuracy (средняя точность по классам)
- Визуализация результатов тестирования

#### 5. Сохранение и загрузка модели

- Сохранение обученной модели в файл `.mlnet` (ZIP-формат)
- Загрузка модели для повторного использования в приложении
- Предсказание на новых текстовых сообщениях

#### 6. Демонстрация предсказаний

- Интерактивный режим тестирования в консольном приложении
- Ввод произвольного текста сообщения пользователем
- Вывод предсказания (спам / не спам)
- Отображение уверенности модели в предсказании

### Результаты

| Тренер                      | MacroAccuracy | Время обучения  |
| --------------------------- | ------------- | --------------- |
| **TextClassificationMulti** | **0,9239**    | **11,7480 сек** |

**Достигнутая точность:** 92,39% (MacroAccuracy)

**Целевая метрика:** требовалось не менее 70%

### Вывод

Разработанная модель успешно решает задачу бинарной классификации текста для определения спам-сообщений. Благодаря использованию подхода TextClassificationMulti, модель способна учитывать контекст сообщения.

### Технологический стек

- **.NET** — платформа разработки
- **ML.NET** — фреймворк машинного обучения
- **ML.NET Model Builder** — инструмент автоматизированного подбора модели
- **TextClassificationMulti** — алгоритм на базе предобученной языковой модели

### Использование

1. Запустите консольное приложение
2. При запросе введите `y` или `д` для тестирования
3. Введите текст сообщения для анализа
4. Получите результат: "Это спам" или "Это не спам" с указанием уверенности модели
5. Для выхода из программы введите `n` или любую другую клавишу при запросе
